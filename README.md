# Email Assistant Chrome Extension

## Project Purpose
The Email Assistant Chrome Extension is designed to enhance your email experience by providing various features that streamline email management, improve organization, and boost productivity. Whether you need to manage your inbox more effectively or automate certain tasks, this extension aims to assist you with seamless and efficient email handling.

## Features
- Smart email sorting and categorization
- Template responses for quick replies
- Reminder and follow-up notifications
- Integration with popular email services

## Setup Instructions
1. **Clone the Repository**:
   Open your terminal and run the following command:
   ```bash
   git clone https://github.com/coemsijopu-ui/copu.git
   ```

2. **Navigate to the Project Directory**:
   ```bash
   cd copu
   ```

3. **Install Dependencies**:
   Make sure you have Node.js installed. Then, run:
   ```bash
   npm install
   ```

4. **Build the Extension**:
   ```bash
   npm run build
   ```

5. **Load the Extension in Chrome**:
   - Open Chrome and navigate to `chrome://extensions/`
   - Enable "Developer mode" (toggle in the upper right corner)
   - Click on "Load unpacked" and select the `dist` folder in your project directory

6. **Usage**:
   Once loaded, you can start using the Email Assistant Chrome Extension by clicking on its icon in the Chrome toolbar. Follow any additional setup prompts if necessary.
